# Mall Cloud 快速开始

## 第零步：配置文件准备

所有服务的 `application.yml` 已提交在仓库中，包含 Nacos 连接信息，可直接使用。
业务配置（数据库、Redis、ES、短信密钥等）全部托管在 Nacos，按需在 Nacos 控制台修改。

> 首次启动前需在 Nacos `7af60364-a045-4561-85a3-3f7a69de938d` 命名空间下确认各服务的 YAML 配置存在。
> Nacos 配置可从 `nacos_config_export_*.zip` 导入。

## 环境检查

| 依赖 | 版本要求 | 检查命令 |
|------|---------|---------|
| JDK | 17+ | `java -version` |
| Maven | 3.8+ | `mvn -v` |
| MySQL | 8.x | `mysql -u root -p -e "SELECT VERSION()"` |
| Nacos | 2.3.2+ | 浏览器打开 `http://localhost:8848/nacos` |
| Redis | 6+ | `redis-cli -h localhost -p 6379 PING` |
| RocketMQ | 5.x | NameServer `localhost:9876` 可连通 |
| Elasticsearch | 7.17 | `curl http://localhost:9200` |
| MongoDB | 5+ | `mongosh localhost:27017 --eval "db.version()"` |
| MinIO | — | 浏览器打开 `http://localhost:9002` |

## 数据库初始化

```sql
-- 单库
CREATE DATABASE IF NOT EXISTS cloud_mall_admin     DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE IF NOT EXISTS cloud_mall_basic     DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE IF NOT EXISTS cloud_mall_product   DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE IF NOT EXISTS cloud_mall_marketing DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE IF NOT EXISTS cloud_mall_member    DEFAULT CHARACTER SET utf8mb4;

-- 分库（订单 8 库）
CREATE DATABASE IF NOT EXISTS cloud_mall_trade_0 DEFAULT CHARACTER SET utf8mb4;
-- ... cloud_mall_trade_1 ~ cloud_mall_trade_7

-- 分库（消息 8 库）
CREATE DATABASE IF NOT EXISTS cloud_mall_message_0 DEFAULT CHARACTER SET utf8mb4;
-- ... cloud_mall_message_1 ~ cloud_mall_message_7

-- 分库（推荐 2 库）
CREATE DATABASE IF NOT EXISTS cloud_mall_recommend_0 DEFAULT CHARACTER SET utf8mb4;
CREATE DATABASE IF NOT EXISTS cloud_mall_recommend_1 DEFAULT CHARACTER SET utf8mb4;
```

## 构建项目

```bash
# 全量构建
mvn clean package -DskipTests

# 增量构建单个模块
mvn clean package -DskipTests -pl mall-gateway -am
```

## 启动顺序

```
外部依赖（先启动）：
  Nacos → MySQL → Redis → RocketMQ → Elasticsearch → MongoDB → MinIO

微服务启动顺序：
  ① mall-gateway (8080)    网关
  ② mall-admin (8030)      认证/权限服务
  ② mall-basic (8022)      基础服务（短信、文件、敏感词）
  ② mall-customer (8025)    C 端用户服务
  ③ mall-product (8023)    商品服务
  ④ mall-order (8026)      订单服务（ShardingSphere）
  ④ mall-pay (8027)        支付服务
  ④ mall-marketing (8024)  营销服务
  ⑤ mall-recommend (8028)  推荐服务（ShardingSphere）
  ⑤ mall-message (8029)    消息服务（ShardingSphere）
  ⑥ mall-admin-bff (8090)  管理后台 BFF
  ⑥ mall-mobile-bff (8091) 移动端 BFF
```

## 验证

```bash
# 1. 检查 Nacos 注册状态（12 个服务全部注册）
# 2. 网关健康检查
curl http://localhost:8080/actuator/health

# 3. 获取图形验证码（白名单）
curl http://localhost:8080/api/admin/v1/auth/getCode

# 4. 测试登录
curl -X POST http://localhost:8080/api/admin/v1/auth/testLogin \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123","uuid":"test","code":"0000"}'

# 5. 保护接口（无 token → 401）
curl http://localhost:8080/api/admin/v1/auth/userDetail

# 6. 保护接口（带 token → 200）
TOKEN="<上一步返回的token>"
curl http://localhost:8080/api/admin/v1/auth/userDetail \
  -H "Authorization: Bearer $TOKEN"
```
