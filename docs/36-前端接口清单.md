# 管理后台前端 API 接口清单

> 按功能域分类，只列前端需要关心的接口
> 写操作统一走 Gateway 直通微服务（不再经过 BFF）
> 2026-07-19

---

## 一、接口分类原则

| 分类 | 说明 |
|------|------|
| **BFF 读聚合** | 前端通过 `/admin/v1/xxx` 调用 BFF，返回 `ApiResult` |
| **直通写操作** | 前端通过 `/api/xxx-api/v1/xxx` 调用 Gateway，直通微服务，返回 `ApiResult` |

不在列表中的接口：字典管理、定时任务、内部 Feign 接口等纯技术接口，前端无需关心。

## 三、Gateway 路由对照

| 路由前缀 | StripPrefix | 目标服务 | 覆盖功能域 |
|----------|:----------:|----------|-----------|
| `/admin/**` | 无 | mall-admin-bff | A-K 所有读查询 |
| `/api/admin-api/**` | 2 | mall-admin-api | A（RBAC 写操作） |
| `/api/product-api/**` | 2 | mall-product-api | B（商品写）C（运营写）H（评价写） |
| `/api/basic-api/**` | 2 | mall-basic-api | D（基础数据写） |
| `/api/message-api/**` | 2 | mall-message-api | E（通知写） |
| `/api/order-api/**` | 2 | mall-order-api | F（订单写） |
| `/api/marketing-api/**` | 2 | mall-marketing-api | G（营销写） |
| `/api/inventory-api/**` | 2 | mall-inventory-api | I（库存写） |
