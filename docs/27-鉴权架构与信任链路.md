> вҡ пёҸ BFF зҡ„ `AuthApiInterceptor` д»…еңЁж”¶еҲ° Gateway йҖҸдј зҡ„ `X-User-*` еӨҙ + `Authorization` еӨҙж—¶еҒҡеҸҢйҮҚж ЎйӘҢгҖӮеҰӮжһң BFF зҡ„жҺ§еҲ¶еҷЁиҮӘе·ұеҸ‘иө· Feign и°ғз”Ёдё”йңҖиҰҒиә«д»ҪпјҢFeignAuthInterceptor иҙҹиҙЈйҖҸдј  Authorization еӨҙгҖӮ

### 2.3 FeignAuthInterceptor йҖҸдј иҜҰи§Ј

```java
// FeignAuthInterceptor.apply()
public void apply(RequestTemplate template) {
    // 1. д»ҺеҪ“еүҚзәҝзЁӢзҡ„иҜ·жұӮдёҠдёӢж–ҮеҸ– Authorization еӨҙ
    RequestAttributes requestAttributes = RequestContextHolder.getRequestAttributes();
    if (Objects.isNull(requestAttributes)) {
        return;  // жІЎжңүиҜ·жұӮдёҠдёӢж–ҮпјҲе®ҡж—¶д»»еҠЎгҖҒMQ ж¶Ҳиҙ№иҖ…зӯүпјү
    }

    ServletRequestAttributes servletRequestAttributes =
            (ServletRequestAttributes) requestAttributes;
    HttpServletRequest request = servletRequestAttributes.getRequest();
    String authorization = TokenUtil.getAuthorization(request);

    // 2. жңү token еҲҷйҖҸдј 
    if (StringUtils.hasLength(authorization)) {
        template.header(AUTHORIZATION, authorization);
    }
    // 3. ж Үи®°дёәеҶ…йғЁиҜ·жұӮ
    template.header(INNER_REQUEST, "true");
}
```
